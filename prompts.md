# Prompts Used During Development

Below are the AI prompts used to assist in building the Kanban Task Board project.

---

## 1. Project Understanding

"Explain how to build a Trello-like Kanban board using React, including components, useState, props, and project structure. Provide step-by-step setup instructions using Vite."

---

## 2. React State & Data Flow

"Help me understand how to store tasks in React state and render them dynamically inside multiple columns using filter and map."

"Explain how to pass data from parent to child using props, and how to update parent state from a child component."

---

## 3. Add / Delete / Move Functionality

"How do I implement task deletion and status updates in React using immutable state updates?"

"Show me how to move a task between columns by updating its status property."

---

## 4. Priority System

"How can I implement a priority system (High, Medium, Low) with dynamic border styling based on priority?"

---

## 5. localStorage Persistence

"How do I persist React state using localStorage so tasks remain after page refresh?"

"What is the best practice for initializing state from localStorage in React?"

---

## 6. Editing Tasks

"How do I implement inline task editing using conditional rendering and controlled input fields in React?"

---

## 7. Drag & Drop (Advanced)

"How do I integrate drag and drop functionality in React using dnd-kit?"

"How can I prevent drag events from interfering with button click events inside draggable components?"

"How do I use a drag handle instead of making the entire component draggable?"

---

## 8. Search & Filtering

"How do I implement a real-time search filter that dynamically filters tasks based on input text?"

---

## 9. UI & Styling

"Help me improve the UI styling to make the Kanban board look modern and similar to Trello."

"How can I structure CSS for a clean column-based layout with responsive behavior?"

---

## 10. Deployment & Version Control

"How do I push my local React project to GitHub?"

"How do I deploy a Vite + React project to Vercel?"

"How do I structure a 3-minute demo presentation for submission?"
